# GreenGrid – Decentralized Community Energy Sharing Platform

## Overview
**GreenGrid** is a web-based user interface that enables communities to monitor, share, and manage renewable energy resources in a decentralized way.  

---

## Tech Stack
- **Vite** 
- **TypeScript**   
- **Tailwind CSS** 
- **Node.js & npm**

---

## Project Setup

### 1️ Clone the Repository
```bash
git clone https://github.com/ruhan-2908/GreenGrid.git

```

### 2 Go Inside the folder 
```bash
cd GreenGrid

```

### 3 Install Dependencies
Make sure you have Node.js (v18 or later) installed.
Then run.
```bash
npm install

```
### 4 Run the Development Server
To start the app locally:
```bash
npm run dev

```

You’ll see output like:
```bash
VITE v5.0.0  ready in 400ms

➜  Local:   http://localhost:8080/

```

Open the shown URL in your browser (e.g., Chrome):
